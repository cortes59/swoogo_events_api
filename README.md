## Tech Stack
- Nodejs
- Express

## Installation
- `git clone https://github.com/cortes59/swoogo_events_api`
- `cd swoogo_events_api`
- `npm install`
- `cp .env.example .env`
- Update the `.env` file with the required environment variables
- `npm run dev`
